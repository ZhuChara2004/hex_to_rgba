## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hex_to_rgba'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hex_to_rgba

## Usage
```
HexToRgba::Converter.convert '72fa52', 0.5
```
